# Use of AI Tools

AI tools were used to support coding, debugging and explanation during this project.

## How AI Was Used

AI helped with:

- generating initial Streamlit code structures
- debugging Python errors
- explaining pandas and Streamlit concepts
- suggesting dashboard features
- improving documentation wording
- structuring the project into stages
- explaining how to use Git, Terminal and deployment tools

## Human Contribution

The project direction, feature selection, testing, interpretation and final presentation were controlled by me.

I focused on understanding:

- how the data flows through the app
- how filters affect the dashboard outputs
- how the coordinate conversion works
- how the screening score is calculated
- what each engineering formula means
- what the model assumptions are
- what the limitations are
- how to explain the project in interviews

## Why This Matters

The aim was not just to generate code, but to use AI as a learning and productivity tool while building a project that I could understand, explain and improve.

This reflects how modern engineering and data work increasingly involves using software tools, automation and AI assistance while still requiring technical judgement and domain understanding.
